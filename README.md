# CompostaFácil

## Sobre o projeto

O **CompostaFácil** é uma calculadora simples de compostagem com visual colorido e educativo.

O usuário informa a quantidade semanal de resíduos orgânicos e o sistema calcula uma estimativa mensal de resíduos aproveitados e de adubo produzido.

Tema:

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

## Funcionalidades

- Calcula resíduos orgânicos por semana.
- Estima resíduos por mês.
- Estima a quantidade de adubo produzido.
- Mostra uma classificação da composteira.
- Apresenta dicas de compostagem.
- Possui visual responsivo e diferente.

## Como usar

1. Baixe o projeto.
2. Abra a pasta no VS Code.
3. Abra o arquivo `index.html` no navegador.
4. Preencha os valores em kg.
5. Clique em **Calcular compostagem**.

## Objetivo educativo

Mostrar que resíduos orgânicos podem ser reaproveitados como adubo natural, reduzindo desperdícios e melhorando a qualidade do solo.
